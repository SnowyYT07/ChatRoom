# ChatRoom In Python

### HOW TO USE

```

Ubuntu@ubuntu:$ git clone https://github.com/SnowyYT07/ChatRoom
Ubuntu@ubuntu:$ cd ChatRoom
Ubuntu@ubuntu:$ python2 server.py 0.0.0.0 5555

in another terminal:

user1@ubuntu:$ cd ChatRoom
user1@ubuntu:$ python2 client.py (ip of "server" machine) 5555

In another terminal:

user2@ubuntu:$ cd ChatRoom
user2@ubuntu:$ python2 client.py (ip of "server" machine) 5555

```
